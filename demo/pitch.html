NullTrace

Private search. Zero browser history.

Your searches never touch Chrome or Firefox history — only encrypted blobs on your machine.

The problem

  • Incognito does not hide searches from the browser or OS
  • History APIs and sync expose what you looked for
  • Users deserve readable history — but only for themselves

Our solution

  • Custom new tab — search never hits browser history API
  • 21 encryption layers — 7 chosen at random per browser session
  • Last 5 searches only — auto-pruned encrypted storage
  • Passphrase unlock — plaintext never persisted

7 of 21 layers (per session)

1AES-256-GCM (PBKDF2-derived key)
2XOR stream obfuscation
3AES-256-GCM (second key)
4Byte reversal
5Shuffled Base64 alphabet
6ROT-7 cipher
7Final AES-256-GCM seal

Data flow

User types query (new tab)
    ↓
Browser encrypts (7 random of 21)
    ↓
Ciphertext → extension storage (max 5)
    ↓
Redirect token → search engine
    ↓
Browser history API: never called
